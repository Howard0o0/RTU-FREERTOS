# 工程目录说明

## Apps
放置上层应用

## Common
FreeRTOS所需要的硬件库

## Debug_Large_Data_Model
IAR生成的中间文件

## Documents
与RTU工程相关的文档

## Driverlib
MSP430F5438A的官方HAL库

## Drivers
自己写的外设驱动

## F5XX_6XX_Core_Lib
MSP430F5438时钟相关的驱动

## Oldproject
老RTU工程的代码

## Source
FreeRTOS源代码
