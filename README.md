# RTU-FREERTOS

# git注意事项
需要忽略的目录:Debug_Large_Data_Model/ ， 如果不忽略，git的时候会产生冲突，并且冲突无法解决

# 代码维护注意事项
1. 使用printf/err_printf/debug_printf时，请在末尾加两个换行(\n\n)，以方便查看串口打印消息
2. MSP430数据类型double默认4字节，可以在Option/General Option/Floating-point中修改


# 工程目录说明

## Apps

## F5XX_6XX_Core_Lib
MSP430F5438时钟相关的驱动

## Oldproject
老RTU工程的代码

## Source
FreeRTOS源代码
